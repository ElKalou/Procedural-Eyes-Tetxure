# Procedural-Eyes-Tetxure
A shader made in Blender that procedurally generates eyes texture


This Blender file contains a shader made in Blender using shader nodes. It allows to procedurally generate eyes texture such as :
![global](https://user-images.githubusercontent.com/43396464/74643814-a3d6ac80-5175-11ea-9f39-128b20356365.png)

To do so, the main node generates various pattern used for the different part of the eyes, based on the user input. Those patterns can then be mixed together to recreate a full eye texture.
